### Реализованные возможности

- Регистрация и вход в свой аккаунт
- Публикация новых постов
    - Добавление картинки в пост
- Система подписок, лайков и комментариев
- Страница рекомендаций
- Лента подписок

### Запуск проекта

1. Склонируйте папку с проектом
2. Скопируйте в папку с проектом файлы *key.txt*, *mail.json*, *secret_key.bin*

### Регистрация

![show registration](/show1.png)

1. Переходим на страницу регистрации
2. Заполняем форму и жмем кнопку
3. Заходим на почту, ищем письмо с подтверждением (может оказаться в спаме), и жмем кнопку "Подтвердить"
4. Поздравляем! Теперь у вас есть аккаунт в нашей социальной сети!
